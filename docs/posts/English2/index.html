我的英语习得之路（二）：正确的英语习得方法

English
Author

Maverick McQuillan

Published

March 1, 2025

我在动笔写作的时候，时间已经进入到了 2025 年的三月份，这个月我大概要吃 60-90顿饭，每顿饭所花费时间大概是 20 分钟，一共是 20-30个小时。如果把吃饭换成别的，比如学数学， 背诵文言文，学一门技术，你会发现这并不容易，如果坚持 20 多年那就更是极其艰难了。但是如果我没有二十多年如一日的每天吃饭，我就不会从一个婴儿长成一个成年人。同样的道理，如果我们的英语能力想从一个婴儿长成成年人，我们也需要至少六年多不间断的“吃饭”。那如何让习得英语的训练变成像吃饭一样的本能，便是我们习得英语能力的关键。

那现在我们来拆解一下《我们为什么会持续不断的吃饭》的问题，这个吃饭的行为大概符合以下的特点：

  1. 如果长时间不吃饭的话，我们会饿，而饿是一个比较痛苦的感受。
  2. 吃饭不仅解决我们饿的感受，还能让我们体会到吃东西的快感。
  3. 我们经常吃不一样的食物（肉、面包、牛奶等），是我们对吃饭这个行为不感到厌烦
  4. 心情难过的时候，我们会选择吃好吃的来解压
  5. 吃饭这个行为不仅通过专门吃正餐来执行，平时我们闲得没事也可以吃一些零食（花费时间比较少,且随时随地可以进行）。

所以我们好的英语习得方法也应该像吃饭一样，符合以下特点：

  1. 随时随地可以习得英语
  2. 要有成就感
  3. 每次习得的时候，即使不开心也不能有抵触心理
  4. 每次习得英语的内容要保持新鲜感
  5. 如果长时间没有接触英文信息会感到心慌

其实如果我们坚持错误的英语习得方法,很难同时满足以上的所有特征,这也就是为什么我们对英语不能上瘾的原因。 经过我长时间的探索，我发现有几个英语习得的方法可以满足以上的特征，能让我们持续高效得提升自己的英语能力，下面我将从阅读和听力两个方面来介绍我的英语习得方法。

阅读

阅读英文小说

我还记得那是在 2021 年年末，我看了一本书叫做《把你的英语用起来》，里面推荐一种英语习得的方法，就是阅读英文书籍，当时我正着急如何 考过英语六级（大学院里规定如果英语六级不过，不让保研），但是我已经考了四五回都是不过，本人也不愿意在用刷题的方式来备考，于是想尝试着用 阅读英文名著的方法来看看，说实话，我并没有对这种英语习得方法报太高的期望，但是还是硬着头皮花了一周多的时间通过隔一个单词一查的方法 把《The Catcher In The Rye》这本小说读完了。读完之后，我干到震撼，因为以我当时的英文水平竟然能读懂60%，而且我当时非常的自豪，心想全 中国这么多大学生，但是能通过阅读英文名著来感受乐趣的人肯定很少，我认为我当时享受到了只有少数人才能有的高级别乐趣（这种乐趣甚至是用钱买不到的）。可能只是阅读一本英文小说对我的英文能力提升的帮助很小，但是却极大得加强了我习得英语的自信心和兴趣。事后看来，这份自信与兴趣是我人生的无价之宝。

总结来讲，阅读英文小说有以下优点：

  1. 英文小说相比于非小说类的书籍来讲更容易让人着迷。（人是喜欢听故事的）
  2. 一个作者在写作的时候会不断重复自己的习惯性用语，这样在阅读的时候你会不断得重复遇到你不认识的生词，从而彻底牢记这个单词。
  3. 好的英文小说在阅读后会让读者回味无穷，这样你不仅获得成就感，更能在精神上体验到快乐，从而更加热爱英语。

我在阅读完《The Catcher In The Rye》之后，想着继续阅读一些英文小说，当时偶然发现了《Animal Farm》这本书，起初我以为是童话书，心想难度 肯定比较低，于是就阅读了，但是读着读着发现这是一本政治讽刺的小说，而且里面所表达的现象跟我们国家还有些像，引发了我比较深刻的思考，读完之后， 我更感慨的一点是我能通过吸取英文信息来思考社会问题，这种感觉是无可比拟的，也是我大脑的一次重大的软件升级。

我在阅读《The Alchemist》时，才开头就碰见商人 merchant 这个单词，这个单词一直在书中反复出现， 导致我现在都不曾遗忘，实际上看英文书籍对词汇量的增长是最有帮助的，如果通过做阅读题，你会发现同一个 单词出现的频率很低，所以这就是我为什么极力反对通过备考的方式来学习英语，第一它磨灭你对英语的兴趣， 第二它的英语习得效率实际上非常低。

学习英文教材

我一直有一个观念就是英语其实是一个可以被潜在习得的技能，它真得不用专门花时间去专门学习，你可以潜移默化的习得，就像我们如何习得中文讲话， 会看中文字一样。

当你通过阅读英文小说建立了自信之后，我非常推崇你去用英语来学习一门技能。

  1. 如果你想了解 AI，就必须要学习一些编程知识，完全可以通过《Python Crash Course, 3rd Edition: A Hands-On, Project-Based Introduction to Programming》这本英文教材来了解 Python
  2. 如果你想学习一些财务知识，可以通过曼昆的《Principles of Economics》去了解什么是通货膨胀、中央银行是怎么运行的、税对市场的影响，从而提升自己的财商。
  3. 如果你想学习数学，可以通过阅读《Algebra》和《Calculus》等美国教材来直接去了解数学的内容和逻辑是什么。

通过阅读英文教材来去学习一门技能是一个一举三得的事情，你在增加英语词汇量的同时，也能锻炼自己的思维逻辑，开阔自己的视野，习得一门技能。 总之，一句话不要学习英语，而是要通过英语来学习你感兴趣的东西。

浏览英文信息

西方世界有很多优质的信息，很多有思想的人会通过建立博客的形式来去发表自己的见解。他们的见解非常深刻，我在阅读他们写的文章时常常非常后悔为什么 没有很早就遇见他们。在浏览英文信息的同时，你也会增强对西方文化的理解，而且西方世界对信息的透明度要远远高于国内，你可以在网上查看很多优质的数据 分析，我将会专门写一篇文章来去介绍我比较喜欢的英文信息来源。

总体来讲，其实阅读英文信息对英语能力的增强效果不如看英文书籍，但是它绝对比刷卷子和看英文教材的效率要高好几遍，并且你习得的是真正的英语，说句实话， 我在新概念一中学到的“pardon”这个单词从来没有出现在任何我观看的 YouTube 视频和英文信息里。

我是在去年的四五月份开始重视我的英语听力能力的发展，也探索了很多方法，虽然目前我的听力能力还是比较 low，但是我认为我找到了正确的方法，只要持之以恒的通过“不坚持”的 方式坚持下去，我肯定能在英语听力和口语方面有着长足的进步。

听力

刷TikTok

我认为对于中国人来讲，如果国家开放 TikTok 软件给我们去刷，我相信整体国人的英文能力会提升一个台阶，因为刷 TikTok 上的英文内容是我目前认为习得英语最高效的方式，它满足我在上面提到的所有 类似于吃饭的特点。

刷 TikTok 有以下的好处：

  1. 不用担心找不到自己感兴趣的内容，TikTok 的算法会自动把我们想看的英文内容推送过来。
  2. 刷 TikTok 很简单，只需要动动手指。
  3. TikTok 上的内容都是非常贴近英文世界的现实情况，特别像我们从小长大的环境一样，在这种情况下，我们习得的是最接地气英文表达。
  4. 刷 TikTok 几乎不费力气，而且随时随地都可以开始。

反复听同一主题的音频

根据著名的“i+1”法则，我们只有在听取稍微高出我们现有英语能力的音频，才能增强英语能力。但是如果真按照这个标准来选择音频的话，我们只能去找动画片来看了。大家都是成年人了， 谁还对动画片感兴趣呢？所以要想获得“i+1”的效果，又要听我们感兴趣的音频，就必须要妥协。怎么妥协？就是重复的去听一段不太长得音频（大概在 15-20 分钟左右），我认为初期必须要 连续听五天，然后才可换下一段同主题的音频，这样逐渐我们就会能听懂很多单词和固定的短语搭配，从而提升自己的英语听力和口语能力。

看YouTube上的情感类视频

我在刷YouTube 视频时，发现那些科研、政治和新闻类的英文视频我很难听懂，但是如果边看英文字幕边去看一些讲 个人情感经历的视频，我就能听得很明白。可能人天生就喜欢看别人的热闹，对情绪的感觉更加敏感，我们可以调用人性 上的特点来帮我们更沉浸式得习得英语能力。
